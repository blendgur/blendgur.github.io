Welcome to

a .blend hosting site for Blender.SE

Blendgur is a website that facilitates .blend file hosting for Blender StackExchange. It chops up the binary data and stores it in a custom PNG metadata chunk (id= blNd). The images are uploaded to StackExchange's Imgur host and their IDs are encoded in the URL of the download link. All of the processing is done on the client with JavaScript, allowing Blendgur to be hosted on GitHub, since there are no server requirements.

Code is public on GitHub
