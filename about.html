Welcome to

a .blend hosting site for Blender.SE

Blendgur is a website that facilitates hosting .blend files for Blender StackExchange. It works by chopping up the binary data and storing it in the metadata of PNG images, then uploading them to StackExchange's Imgur host. All of the processing is done in JavaScript on the client, so there is no server processing the files.

Code is public on GitHub
